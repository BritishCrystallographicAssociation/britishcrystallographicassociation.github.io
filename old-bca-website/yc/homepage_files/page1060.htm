X-rayman Forum

Web forum organized by Dr Jon Warren, providing a web space for any crystallographic questions/problems and solutions to be discussed. This forum is not sponsored by any particular manufacturer so discussions can be free from bias. Also questions can be posted at any level - if you have wondered what the answer is somebody else will have as well.
