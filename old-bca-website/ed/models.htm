Cut out models to make for yourself

Make your own models

All you need to construct these models for yourself is a computer and a printer. There are several different kinds:

  • Click here for an introduction to space filling shapes or here for a shape to make for yourself, a model of a garnet crystal.
  • Click here for instructions for making more models; these are also space filling shapes, include examples of all 7 of the crystal classes, and how to make the shapes either closed or as boxes with an opening side.
  • A Platonic mobile. Download complete instructions for making a mobile showing all the Platonic solids, from here as a PDF file.
  • Calcite makes an interesting Christmas tree decoration
  • Try finding the symmetry axes in this puzzle page ( PDF format size 5K) with views of crystal forms. You may like to make yourself some paper models of these using the cutout-pages from here. ( PDF format size 18K)
  • What else would you like?
Most of these models were made using the program !PolyNet which allows you to create a planar net from a 3D solid. Click here for details
