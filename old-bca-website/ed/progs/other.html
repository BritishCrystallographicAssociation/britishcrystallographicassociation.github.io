BCA Academic Software Page

Educational Software

This page gives access to information about academic software, mostly public domain, some educational, available on the World Wide Web, not via our server. Please send news of any others to add to this page to the BCA WebMaster ( BCA@ISISE.RL.AC.UK)
Commercial programs are listed elsewhere.
There is also a list of programs for micros.

Programs are listed alphabetically by name:
  • Mineral Web a UK academic site where some mineral crystal structures may be viewed
  • ORTEP III now available from ORNL in a version using PGPLOT graphics, as source, and as executable code for the PC and Mac, Acorn version available from HENSA code e102. ORTEP was written by Carroll K.Johnson and Michael N.Burnett. PGPLOT is a portable Fortran graphics utilties written by Tim Pearson
  • ORTEX a PC version of ORTEP from Galway University
  • RasMol protein molecule plotting, for PCs and Macs, now available from Hensa for Acorn micros
  • Stefen Weber software, including 'Polyhedra','Periodic table viewer','Tiling generator' for quasi crystal symmetries, Diffraction and Laue pattern generators. Recently (1999) updated to include Java applications
  • The Glasgow Software Source
